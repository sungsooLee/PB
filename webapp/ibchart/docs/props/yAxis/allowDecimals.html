allowDecimals (yAxis properties)

y축의 눈금에 소수점을 표시합니다.

Info

name type required(default) description
allowDecimals boolean 선택(false) y축에 소수점 표시 여부

Example

myChart.setOptions({
  chart: {
    type: "line"
  },
  yAxis: {
    title: {
      text: "단위(℃)",
      align: "low",
      rotation: 0,
      style: {
        color: "#575757",
        fontSize: "10px"
      }
    },
    allowDecimals: true // y축의 소수점 표시
  }
}, {
  append: true,
  redraw: true
});
