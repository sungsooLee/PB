lineColor (yAxis properties)

y축 라인의 색상을 설정 합니다.

Info

name type required(default) description
lineColor color 선택(#ccd6eb) y축 라인의 색상

Example

myChart.setOptions({
  chart:{
    type: "column"
  },
  yAxis: {
    lineWidth: 5,
    lineColor: "#FF4500" // y축 라인의 색상 설정
  },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [30,50,90,70]
  },{
    name: "부산",
    data : [40,70,30,60]
  }]
}, {
  append: true,
  redraw: true
});
