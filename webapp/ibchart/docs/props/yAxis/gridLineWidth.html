gridLineWidth (yAxis properties)

그리드 라인의 너비를 설정 합니다.

Info

name type required(default) description
gridLineWidth number 선택(1) 그리드 라인의 너비

Example

myChart.setOptions({
  chart: {
    type: "column"
  },
  yAxis: {
    gridLineWidth: 2 // 그리드 라인의 너비
  },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [30,50,90,70]
  },{
    name: "부산",
    data : [40,70,30,60]
  }]
}, {
  append: true,
  redraw: true
});
