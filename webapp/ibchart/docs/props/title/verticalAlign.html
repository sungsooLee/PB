verticalAlign (title properties)

제목의 상하 정렬 값을 설정합니다.

Info

name type required (default) description
verticalAlign string 선택(center) 상하 정렬 값

Enum

  • verticalAlign
value description
top 상단 정렬
middle 중앙 정렬
bottom 하단 정렬

Example

myChart.setOptions({
  title: {  // title 설정
    text: "2014년 1/4분기",
    verticalAlign: "middle"
  }
}, {
  append: true,
  redraw: true
});
