padding (legend properties)

범례 상자 안의 padding 을 설정합니다.

Info

name type required (default) description
padding number 선택(8) 범례 상자 안의 padding 설정

Example

myChart.setOptions({
  legend: {
    padding: 50 // 범례 상자 안의padding 값 설정
  },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  }]
}, {
  append: true,
  redraw: true
});
