symbolHeight (legend properties)

범례 아이콘의 높이를 설정합니다.
symbolHeight 가 선언되지 않은 경우 아이콘의 높이는 범례 항목의 글꼴 크기로 기본 설정됩니다.
참고 : 범례 아이콘이 line 차트같이 선이나 scatter 차트같이 점으로 표시되는 경우 symbolHeight 설정이 불가합니다.

Info

name type required (default) description
symbolHeight number 선택(undefined) 범례 아이콘의 높이

Example

myChart.setOptions({
  chart: {
    type: "column"
  },
  legend: {
    symbolHeight: 10 // 범례 아이콘의 높이 설정
  },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [30,50,90,70]
  },{
    name: "부산",
    data : [40,70,30,60]
  }]
}, {
  append: true,
  redraw: true
});
