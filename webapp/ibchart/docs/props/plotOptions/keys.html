keys (plotOptions properties)

데이터 배열의 키를 지정합니다.

Info

name type required description
keys Array 선택 데이터 배열의 키 지정

Example

myChart.setOptions({
  chart: {
    type: "dependencywheel"
  },
  plotOptions: {
    dependencywheel: {
     keys: ["from", "to", "weight"], // 데이터 배열의 키 지정
   }
 },
  series: [{
    data : 
      ['한국', '중국', 6225],
      ['한국', '홍콩', 5287],
      ['한국', '베트남', 1628]
  }]
}, {
  append: true,
  redraw: true
});
