animation (plotOptions properties)

시리즈 영역의 애니메이션 효과 사용 여부를 설정합니다.

Info

name type required description
animation boolean 선택(true) 애니메이션 효과 사용 여부

Enum

  • animation
name type required description
duration number 선택(1000) 애니메이션 지속 시간(밀리초)

Example

myChart.setOptions( {
  series: {
    animation: {
      duration: 2000 // 애니메이션 지속 시간
    }
  }
});
