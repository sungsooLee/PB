colorSort (plotOptions properties)

Wordcloud 차트의 색 지정 방식을 설정하는 속성입니다. only wordcloud

Info

name type required description
colorSort string 선택(none) 색지정 방식

Enum

  • colorSort
value description
none 데이터 입력 순서
rank 입력된 데이터의 최댓값부터 내림차순

Example

//wordcloud 차트 설정
myChart.setOptions({
	chart: {
		type: "wordcloud"
	},
	plotOptions: {
		wordcloud: {
			colorSort: "rank"
		}
	}
}, {
	append: false
});

//wordcloud 데이터 로드
myChart.loadSearchData({
	"ibchart": {
		"data": [{
			"series": [{
				"seriesName": "WordCloud",
				"pointName": "IBLeaders",
				"value": 40
			}]
		}, {

				"value": 50
			}]
		}, {
