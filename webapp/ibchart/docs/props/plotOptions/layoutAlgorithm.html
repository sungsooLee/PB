layoutAlgorithm (plotOptions properties)

배치 알고리즘을 설정합니다.
참고 : tilemap 에서는 타일의 모양을 설정하는 속성입니다.

Example

myChart.setOptions({
  plotOptions: {
    packedbubble: {
      layoutAlgorithm:{ // 배치 알고리즘 설정
        splitSeries: false,
        bubblePadding:10,
        gravitationalConstant: 0.02
      }
   }
 },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [10,15,40,40]
  },{
    name: "부산",
    data : [50,40,20,70]
  }]
}, {
  append: true,
  redraw: true
});
