pointPlacement (plotOptions properties)

포인트의 위치를 설정 합니다.

Info

name type required description
pointPlacement string 선택 포인트 위치

Enum

  • pointPlacement
value description
on 축의 점 위치
between 축의 점간 사이 위치

Example

myChart.setOptions({
  plotOptions: {
    column: {
      pointPlacement:"between"    //컬럼차트 속성은 포인트와 포인트 사이
    },
    line : {
      pointPlacement:"on"         //라인차트 속성은 x축 포인트 위에
    }
 },
  series: [{
    type: "column",
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [30,50,90,70]
  }]
}, {
  append: true,
  redraw: true
});
