minLinkWidth (plotOptions properties)

링크 선의 최소 너비를 설정합니다.
sankey 차트, dependencywheel 차트에서만 사용됩니다.

Info

name type required description
minLinkWidth number 선택(0) 링크 선의 최소 너비

Example

myChart.setOptions({
  plotOptions: {
    sankey: {
      minLinkWidth: 5 // 노드를 잇는 링크 선의 최소 너비 설정
   }
 },
  series: [{
    data:[
      ["node1","node2",20],
      ["node1","node3",5],
      ["node4","node2",3],
      ["node5","node1",10]
    ]
  }]
}, {
  append: true,
  redraw: true
});
