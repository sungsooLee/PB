fillColor (plotOptions properties)

시리즈 영역의 배경색을 설정 합니다.

Info

name type required description
fillColor color 선택 시리즈 영역의 배경색

Example

myChart.setOptions({
  chart: {
    type: "area"
  },
  plotOptions: {
    area: {
     fillColor:"#ffc0cb" // 시리즈 영역의 배경색
   }
 },
  series: [{
    data : [60,40,20,80]
  }]
}, {
  append: true,
  redraw: true
});
