states (plotOptions properties)

시리즈의 상태 속성을 설정 합니다.

Info

name type required description
states object 선택 시리즈의 상태 속성
states.hover object 선택 마우스 hover 시 속성

Enum

  • states.hover Info
value type required description
enabled boolean 선택(true) 사용 여부
borderColor color 선택 테두리 색상
borderWidth number 선택(1) 테두리 너비
lineWidth number 선택(2) 라인 너비
(시리즈 형태가 선의 형태일 때 사용)
brightness number 선택(0.1) 기본 색상 기준 밝기 정도
color color 선택 hover 시 색상
brightness 과 동시에 설정 시 color 가 우선 적용됩니다.

Example

myChart.setOptions({
  plotOptions: {
    bar: {
      states: {
        hover: { // 상태가 hover 일 때 설정
          color: "#ff0000",
          borderColor:"#000000",
          borderWidth: 3
        }
      }
    }
  },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [30,50,90,70]
  },{
    name: "부산",
    data : [40,70,30,60]
  }]
}, {
  append: true,
  redraw: true
});
