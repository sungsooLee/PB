lineWidth (plotOptions properties)

시리즈 선의 너비를 설정합니다.

Info

name type required description
lineWidth number 선택(2) 시리즈 선의 너비

Example

myChart.setOptions({
  plotOptions: {
    line: {
      lineWidth: 5
   }
 },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  }]
}, {
  append: true,
  redraw: true
});
