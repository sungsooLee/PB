stacking (plotOptions properties)

시리즈의 데이터 stack 방법을 설정 합니다.

Info

name type required description
stacking string 선택 stack 방법

Enum

  • stacking
value description
null 사용 안함
normal 일반 stack 방법
percent 비율 stack 방법

Example

myChart.setOptions({
  chart: {
    type: "column"
  },
  plotOptions: {
    column: {
      stacking: "normal" // stack 방법 설정
    }
 },
  series: [{
    name: "서울",
    data : [60,40,20,80]
  },{
    name: "인천",
    data : [30,50,90,70]
  },{
    name: "부산",
    data : [40,70,30,60]
  }]
}, {
  append: true,
  redraw: true
});
