nodes (plotOptions properties)

개별 노드에 대한 속성을 정의합니다.
sankey 차트, dependencywheel 차트에서만 사용이 가능합니다.

Info

name type required description
nodes Array[object] 선택 개별 노드 속성
nodes.color color 선택 노드의 색상
nodes.column number 선택 노드를 배치할 위치에 대한 열 인덱스
nodes.dataLables object 선택 상태 속성
nodes.height number 선택 노드의 높이
nodes.id string 선택 노드의 ID
선언하지 않으면 데이터의 from나 to 를 참조해서 자동 생성합니다.
nodes.offset number or string 선택(0) 노드의 상하 위치를 지정

Example

myChart.setOptions({
  plotOptions: {
    sankey: {
		  keys: ['from','to','weight'], 
      nodes: [ // 각 노드 속성 설정
        {
          id:"node1",
          color:"#ff0000",
          column:2,
          dataLabels:{
            enabled:false
          }
        },
        {
          id:"node2",
          color:"#ff0000",
          column:3,
          offset:-20
        },
      ]
   }
 },
  series: [{
data:[
      ["node1","node2",20],
      ["node1","node3",5],
      ["node4","node2",3],
      ["node5","node1",10]
    ]
  }]
}, {
  append: true,
  redraw: true
});
