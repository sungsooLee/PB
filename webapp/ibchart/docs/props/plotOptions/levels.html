levels (plotOptions properties)

데이터 레벨 별 속성을 지정합니다.

Info

name type required description
levels object 선택 데이터 레벨 별 속성 지정
  • levels Info
name type required description
level number 필수 속성을 설정할 레벨 지정 (1부터 시작해서 자식마다 +1 됩니다.)
layoutAlgorithm string 선택 배치 알고리즘 설정
borderColor color 선택 테두리 색상
borderWidth number 선택 테두리 너비
color color 선택 색상
참고 : colors 나 colorAxis 속성이 우선됩니다.
dataLabels object 선택 데이터 레이블 설정

Example

myChart.setOptions({
  chart: {
    type: "treemap"
  },
  plotOptions: {
    treemap: {
     levels: [ // 레벨 별 속성 설정
        {
          level: 2,
          dataLabels: {
            enabled: true,
            color: "#ffffff"
          }
        },
        {
          level: 1,
          borderWidth: 8,
          dataLabels: {
            enabled: true,
            backgroundColor: "#ffffff"
          }
        },
     ]
   }
 },
  series: [{
    data : [
      {
        id: '0.0', // level : 1
        parent: '',
        value : 30,
        name: 'A'
      },
      {
        id: '1.1', // level : 2
        parent: '0.0',
        value : 20,
        name: 'a-1'
      },
      {
        id: '1.2', // level : 2
        parent: '0.0',
        value : 10,
        name: 'a-2'
      },
      {
        id: '0.1', // level : 1
        parent: '',
        value : 10,
        name: 'B'
      },{
        id: '2.1', // level : 2
        parent: '0.1',
        value : 10,
        name: 'b-1'
      }
    ]
  }]
}, {
  append: true,
  redraw: true
});
