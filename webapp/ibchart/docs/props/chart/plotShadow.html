plotShadow (chart properties)

Plot 영역의 그림자 효과 여부를 설정 하는 속성입니다.

Info

name type required(default) description
plotShadow boolean 선택(false) 차트 영역에 대한 그림자 표시 여부

Example

//차트 기본 속성 설정
myChart.setOptions({
  chart: {
    plotShadow : true
  },
  series: [{
    data : [60,40,20,80]
  }]
}, {
  append: true,
  redraw: true
});
