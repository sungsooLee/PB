useHTML (subtitle properties)

부제목 문자열에 html tag 사용 여부를 설정합니다.

Info

name type required description
useHTML boolean 선택(false) 부제목 문자열에 html tag 사용 여부

Example

myChart.setOptions({
  title: {  // title 설정
    text: "cc별 중고자동차 판매량"
  },
  subtitle: {  // subtitle 설정
    text: "<span style='font-size: 1.2em; font-weight: bold; '>2014년</span> 1/4분기",
    useHTML: true
  }
}, {
  append: true,
  redraw: true
});
