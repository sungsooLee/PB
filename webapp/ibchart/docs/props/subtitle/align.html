align (subtitle properties)

부제목의 좌우 정렬 방법을 설정합니다.

Info

name type required (default) description
align string 선택(center) 좌우 정렬 값

Enum

  • align
value description
center 중앙 정렬
left 좌측 정렬
right 우측 정렬

Example

myChart.setOptions({
  title: {  // title 설정
    text: "cc별 중고자동차 판매량"
  },
  subtitle: {  // subtitle 설정
    text: "2014년 1/4분기",
    align: "left"
  }
}, {
  append: true,
  redraw: true
});
