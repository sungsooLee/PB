y (subtitle properties)

부제목의 상하 포지션 이동값을 설정합니다.

Info

name type required (default) description
y number 선택(0) 부제목 상하 포지션 이동 값

Example

myChart.setOptions({
  title: {  // title 설정
    text: "cc별 중고자동차 판매량"
  },
  subtitle: {  // subtitle 설정
    text: "2014년 1/4분기",
    y: 25
  }
}, {
  append: true,
  redraw: true
});
