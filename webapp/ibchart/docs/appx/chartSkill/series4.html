시리즈 관련기능 4 (chartSkill)

시리즈 관련기능 slicedOffset 을 구현 할 수 있습니다. pie 이격 거리

Example

myChart.setOptions({
  chart:{
    type:'pie'
  },
  plotOptions: {
    series: { //series 대신 pie를 넣어도 상관없음.
      slicedOffset: 20
    }
  },
  series: [{
    data : [{
      name: "축구",
      y: 44.2,
      selected: true,
      sliced: true
    },
    ["농구",26.6],
    ["배구",20],
    ["야구",3.1],
    ]
  }]
},{
  append: true,
  redraw: true
});
