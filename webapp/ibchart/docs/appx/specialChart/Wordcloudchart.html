Wordcloud chart (special Chart)

WordCloud 차트는 가중치(값)에 따라 단어의 크기, 위치, 색을 지정하고 단어를 시각적으로 표현하는 차트입니다.
단일 시리즈이며 그 한 개의 시리즈에 단어와 가중치 데이터로 구성되어 있습니다.
제약사항 차트를 생성할때, opt.plugin값을 d3로 설정하고 사용해야 하고 opt.width와 opt.height값은 300px이상으로만 설정해야 합니다.(※너무 작을경우 차트가 표현될 수 없음) %로 설정되었을때 부모의 너비 / 높이가 300px 보다 작게 값을 설정하지 않도록 해야 합니다.

Example

//wordcloud 차트 생성
createIBChart(chart,"myChart", {
	width: "500px",
	height: "500px",
	plugin: "d3"
});

//wordcloud 차트 설정
myChart.setOptions({
	chart: {
		type: "wordcloud"
	}
}, {
	append: false
});
