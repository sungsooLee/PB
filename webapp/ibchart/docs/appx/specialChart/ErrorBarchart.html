ErrorBar chart (special Chart)

ErrorBar 차트를 표현합니다. highcharts-more.js 필요

Example

myChart.setOptions({
  chart:{
    type:'errorbar'
  },
  series: [{
    name: 'Temperatures',
    data: [[6, 8], [5.9, 7.6], [9.4, 10.4], [14.1, 15.9], [18.0, 20.1], [21.0, 24.0]]
  }]
},{
  append: true,
  redraw: true
});
