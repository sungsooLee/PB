ColumnRange chart (special Chart)

ColumnRange 차트를 표현합니다. highcharts-more.js 필요

Example

myChart.setOptions({
  chart:{
    type:'columnrange',
    inverted:true // 축 반전
  },
  series: [{
    name: 'Temperatures',
    data: [
      [-19.7, 9.4],
      [-12.7, 16.5],
      [-30.5, -9.4],
      [-1.4, 19.9],
      [5.0, 22.6],
      [2.9, 29.5],
      [19.2, 30.7],
      [-7.3, 6.5]
    ]
  }]
},{
  append: true,
  redraw: true
});
