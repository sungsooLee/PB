TileMap chart (special Chart)

TileMap 차트를 표현합니다. modules/heatmap.js 필요 modules/tilemap.js 필요
※ 반드시 heatmap.js, tilemap.js 순으로 import 해야 합니다.

Example

myChart.setOptions({
  chart: {
    type: 'tilemap',
  },
  plotOptions: {
    tilemap: {
      layoutAlgorithm: 'hexagon', // 타일의 모양을 육각형으로 설정
    },
  },
  series: [{
    data: [
      [3, 3, 'name1'], 
      [4, 3, 'name2'],
      [5, 4, 'name3'],
      [6, 3, 'name4'],
      [7, 3, 'name5'],
      [6, 2, 'name6'],
      [5, 2, 'name7'],
      [4, 2, 'name8']
    ]
  }]
},{
  append: true,
  redraw: true
});
