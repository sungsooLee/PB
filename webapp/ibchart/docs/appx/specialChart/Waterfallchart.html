Waterfall chart (special Chart)

Waterfall 차트를 표현합니다. highcharts-more.js 필요

Example

myChart.setOptions({
  chart:{
    type:'waterfall'
  },
  series: [{
      upColor: '#FF6666', //증가 시 표시되는 색
      Color: '#448EF6', // 감소 시 표시되는 색
      data: [120, 569, -131, 231, -342, 233, 231, -42, 103, -131, -255, 98]
  }]
},{
  append: true,
  redraw: true
});
