addSeries (ChartObject method)

시리즈를 차트에 동적으로 추가 합니다.

Syntax

{ibchart}.addSeries(prop, [opt]);

Parameters

Name Type Required Description
prop object 필수 시리즈 속성 및 데이터
opt object 선택 옵션 속성
opt.redraw boolean 선택 설정 후 다시 그릴지 여부

Returns

none

Example

myChart.addSeries({
  data: [10, 20, 30]  // 시리즈의 데이터 설정
});

Since

version desc
