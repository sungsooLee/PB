loadSearchData (ChartObject method)

인자로 받은 데이터 객체 또는 문자열을 차트에 로드 합니다.

Syntax

{ibchart}.loadSearchData (data, [opt]);

Parameters

Name Type Required Description
data object or string 필수 로드할 데이터 객체 또는 문자열
opt object 선택 옵션 속성
opt.append boolean 선택(false) 데이터 병합 여부
opt.resetData boolean 선택(true) 데이터 병합 여부(속성을 유지하면서 데이터 초기화 여부, opt.append값과 opt.mergeData값이 false이면 데이터 초기화)
opt.mergeData boolean 선택(false) 데이터를 기존 데이터에 덮어씌울지 여부(opt.resetData값이 false일때만 적용됨)
opt.deepCopy boolean 선택(false) 깊은 복사(데이터량에 따라 시간이 더 필요할 수 있음)

Returns

none

Example

// 데이터를 차트에 로드 처리
myChart.loadSearchData({
  ibchart: {
    data: [{
      series: [1,10,100]
    },{
      series: [2,20,200]
    },{
      series: [3,30,300]
    }]
  }
});
